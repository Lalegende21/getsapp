permissions utilisateur

Role Visisteur

Cet utilisateur aura ces autorisations:

  • Consulter les informations

Cet utilisateur n'aura pas ces autorisations:

  • Ajouter de nouvelles données
  • Modifier les données
  • Supprimer des données
  • Modifier les autorisations d'autres utilisateurs
